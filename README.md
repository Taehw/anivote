# 🏆 Anivote (애니vote)

애니메이션 명장면을 감상하고, 당신의 '최애' 장면에 투표하세요!
Next.js와 Supabase를 활용해 빠르게 검증(MVP)하기 위해 개발된 사이드 프로젝트입니다.

## 🛠 Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Database:** Supabase (PostgreSQL)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## ✨ Key Features
- **투표 기능:** 사용자는 마음에 드는 움짤에 실시간으로 투표할 수 있습니다.
- **랭킹 시스템:** 투표 수에 따라 실시간으로 정렬된 순위를 확인할 수 있습니다.
- **서버리스 아키텍처:** 별도의 백엔드 서버 구축 없이 Supabase를 통해 DB와 통신합니다.
